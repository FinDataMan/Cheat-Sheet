HTML Tags

Introduction

HTML provides structure to the content appearing on a website, such as images, text, or videos.

Anatomy

HTML is composed of elements. These elements structure the webpage and define its content:

  • An opening tag <tag name>
  • Content displayed
  • A closing tag </tag name>

HTML Tags CheatSheet
Atribute name <tag name> Description
Headings <h1> ... <h6> Used to provide titles for sections of content.
Divs <div> Container that divides the page into sections.
Paragraphs <p> Contains a block of plain text.
Span <span> Used to separate small pieces of content that are on the same line as other content.
Italics <em> Will generally render as italic emphasis.
Bold <strong> Will generally render as bold emphasis.
Line Breaks <br> Modifies the spacing in the browser.
Unordered List <ul> Creates a list of items in no particular order, outlined with bullet points.
Ordered List <ol> Creates a list of ordered items, outlined with numbers.
Image <img> Self closing tag that allows you to add an image to a web page.
Video <video> Tag that allows you to display videos.
